# Theme Notes

## Current Direction

The active theme direction is railway / track-laying.

Why this is the stronger default:

- The core mechanic is path-building, which already reads more like laying track than moving a creature.
- The tile shapes map cleanly to track geometry: straight, soft turn, and hard turn.
- The player is extending a single continuous route toward a destination, which fits a railway fantasy naturally.

Current terminology direction:

- Path: railway / track
- Leading end: engine
- Tiles: track tiles
- Resource: token

`Surveyor` is now just flavor for the spend action, not the resource name:

- Spend 1 token to hire a surveyor and reveal 2 additional track tiles.
- Collecting a token grants +1 token.

This reads as scouting ahead for better route options before committing.

## Snake / Apple Theme Record

The original theme was snake-based.

What was strong about it:

- `Apples` are an immediate callback to the classic snake game.
- The game already uses collision, self-trapping, and a single advancing front, which overlap with snake expectations.
- The apple theme is simple and legible for players without needing more explanation.

What was weaker about it:

- The actual board play is about constructing a path tile by tile, not steering a free-moving creature.
- The tile geometry feels more structural than animal-like.
- As the design leans into route planning and forced placements, the railway metaphor carries the mechanic more directly.

Conclusion:

- Snake/apple is a valid alternate theme and worth keeping in the project history.
- Railway/survey is the better fit for the current rules as written.

## Other Resource Names Considered

- `Switch`: excellent spoken language and kid readability, but it suggests branching track or junctions.
- `Signal`: train-native, but less natural as a token you spend.
- `Blueprint`: solid board-game tone, but more abstract for kids.
- `Track`: easy to understand, but it implies raw material rather than improved options.
