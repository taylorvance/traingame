## Next Step

Build a browser prototype of the core game loop:

- Draw 2 track tiles
- Optionally spend 1 token to hire a surveyor and reveal 2 more
- Choose 1 forced placement
- Advance the engine
- Collect tokens, check win/loss, repeat
